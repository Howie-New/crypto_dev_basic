# 精通比特币（第二版）

**Mastering Bitcoin 2nd Edition - Programming the Open Blockchain**

![Mastering Bitcoin 2nd Edition](images/cover.png)

## 关于本仓库

本仓库原书是 [original english version](https://aantonop.com/books/mastering-bitcoin/) 

## 原书信息

- 原书官网: [bitcoinbook.info](https://bitcoinbook.info/)
- 原书购买: [Amazon](https://www.amazon.com/Mastering-Bitcoin-Programming-Open-Blockchain/dp/1491954388)

- 中文翻译原仓库: [tianmingyun/MasterBitcoin2CN](https://github.com/tianmingyun/MasterBitcoin2CN)

我对这个中文翻译版本[精通比特币(第二版)--区块链编程](https://github.com/inoutcode/bitcoin_book_2nd)
做了一些修改, 初心是在当前Age of AI, 学习新技术的效率已经有了巨大的提升, 本文档的学习方法完全可以在Claude Code, Codex等coding agent的协助下快速学习并写一些demo来深刻理解区块链的原理

本书主要面向开发人员，前两章对比特币的介绍也适用于非开发人员。任何对技术有基本了解的人都可以阅读前两章，以深入了解比特币。
