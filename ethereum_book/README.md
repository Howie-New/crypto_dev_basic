# 精通以太坊

**Mastering Ethereum - Building Smart Contracts and DApps**

![Mastering Ethereum](images/cover.png)

## 关于本仓库

本仓库原书是 [Mastering Ethereum](https://github.com/ethereumbook/ethereumbook)
我对这个中文翻译版本[精通以太坊--实现数字合约](https://github.com/inoutcode/ethereum_book)做了一些修改, 初心是在当前Age of AI, 学习新技术的效率已经有了巨大的提升, 本文档的学习方法完全可以在Claude Code, Codex等coding agent的协助下快速学习并写一些demo来深刻理解区块链的原理

本书主要面向开发人员，前两章对以太坊的介绍也适用于非开发人员。任何对技术有基本了解的人都可以阅读前两章，以深入了解以太坊。
